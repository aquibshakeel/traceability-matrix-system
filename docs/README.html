üöÄ Quick Start

  1. Install
    npm install npm run build
  2. Set AI Provider
    # Claude (recommended) export CLAUDE_API_KEY="sk-ant-your-key" # Or OpenAI export OPENAI_API_KEY="sk-your-key" export AI_PROVIDER="openai"
  3. Generate AI Scenarios
    npm run generate

    Creates baseline scenarios in .traceability/test-cases/ai_cases/

  4. Analyze Coverage
    # Using config file npm run continue # Or external repos (CLI) npm run continue -- customer-service \ --service-path=/path/to/service \ --baseline-path=/path/to/baseline.yml
    Output: Premium HTML report at .traceability/reports/{service}-report.html

üéØ Common Workflows

Daily Development

# 1. Make changes, add/update tests # 2. Run analysis npm run continue # 3. Review report # 4. Fix P0 gaps if any

QA Workflow

# 1. Review AI-generated scenarios cat .traceability/test-cases/ai_cases/service-ai.yml # 2. Create baseline from approved scenarios # 3. Run analysis npm run continue # 4. Report gaps to dev team

CI/CD Integration

npm run continue # Check P0 gaps P0_GAPS=$(jq '.summary.p0Gaps' .traceability/reports/service-report.json) if [ "$P0_GAPS" -gt 0 ]; then echo "‚ùå P0 gaps detected" exit 1 fi

‚ö° Performance Tips

  • First run: 2-5 minutes (AI generation)
  • Subsequent runs: 1-2 minutes (matching only)
  • Parallel analysis: Run services independently
  • Cache: AI responses cached for 24 hours
