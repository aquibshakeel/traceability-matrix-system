The Core Question

"How does the system know which unit test belongs to which business scenario?"

Answer: 7 Advanced Matching Strategies

Strategy Weight What It Does
Exact Matching 2.0 Direct string match (API endpoints, IDs)
Fuzzy Matching 1.5 Token-based similarity (handles typos)
Semantic Matching 1.2 Synonym expansion (create=add=insert)
Keyword Matching 1.0 Domain keywords (POST, 404, customer)
Levenshtein - Edit distance (catches typos)
Jaccard - Set-based word similarity
Regex - Pattern matching (explicit references)
