Add AI Provider

1. Create provider:

export class GeminiProvider implements AIProvider {
  async generateTestScenarios(api: API) {
    // Implementation
  }
}

2. Register:

// lib/ai/AIProviderFactory.ts
case 'gemini':
  return new GeminiProvider(config)

ğŸ› Debugging

Common Issues:

  • AI not matching: Check prompts in verbose mode
  • Tests not found: Verify test directory and pattern
  • Reports not generated: Check output directory permissions
